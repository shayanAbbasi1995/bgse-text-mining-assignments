# BGSE Text Mining Assignments

Jupyter notebooks from the Text Mining course at the Barcelona School of Economics (BGSE).

## Contents

| Notebook | Topic |
|---|---|
| `Working with Text Assignment 1.ipynb` | Text preprocessing — tokenization, stemming, TF-IDF |
| `Classifying Text Assignment 2.ipynb` | Document classification using Naive Bayes and logistic regression |
| `Word2Vec Assignment 3.ipynb` | Word embeddings with Word2Vec — training and semantic similarity |
| `Text_Mining_Project.ipynb` | Final project — end-to-end NLP pipeline on a real-world dataset |
